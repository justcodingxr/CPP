1.泛化编程，是对抽象算法的编程，泛化指的是可以广泛适用于各种数据类型。

2.模板是实现泛化编程的主要方法之一，特点是参数不仅仅是一般的数值形式，还可以是一种类型。
可以避免重载函数时函数体重复设计。

3.模板函数
实例化的模板

4.函数重载与函数模板
重载函数各个函数体内可以执行不同的代码，但同一个函数模板实例化的不同模板函数都执行相同的代码，只是数据类型有差别。
调用顺序：
	先一般函数，再找模板
	
	
1.函数模板：
	template <模板形参> 函数类型 函数名（函数形参）{
		//
	}

2.类模板：
	template <模板形参> 
	calss 类名{
	//类体
	};
	
大多数编译器要求:类模板的声明与实现放同一个文件；
